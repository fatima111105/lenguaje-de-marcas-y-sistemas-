¿Qué es HTML?

HTML es el lenguaje de marcado que se utiliza para crear páginas web. Define la estructura y el contenido de una página mediante etiquetas.

Características principales

  • Lenguaje de marcado, no de programación
  • Define la estructura de una página web
  • Utiliza etiquetas para organizar el contenido
  • Es interpretado por los navegadores web
